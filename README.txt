# Comparador de Precios Uruguay - Brasil 🇺🇾 🇧🇷

## Descripción

Este proyecto permite comparar precios de productos básicos entre supermercados de Uruguay y Brasil.
Está orientado a ayudar a los residentes de zonas fronterizas (como Artigas y Quaraí) a optimizar sus compras.

Toda la información es pública, proveniente de SIPC, supermercados online y publicaciones oficiales.

---

## Instalación en local

1. Clonar el repositorio o copiar los archivos en tu PC.
2. Instalar Node.js (si no lo tenés).
3. Desde la carpeta del proyecto correr:

    ```bash
    npm install express
    ```

4. Levantar el servidor:

    ```bash
    node server.js
    ```

5. Acceder a:

    ```bash
    http://localhost:3000
    ```

## Funcionalidades

- 🛒 Comparación de precios URU/BRL.
- 🧠 Valor del Real ajustable manualmente (guardado en LocalStorage).
- 🔍 Buscador de productos en tiempo real.
- ✍️ Panel de Administración para editar la canasta.
- 🛡️ Modal informativo la primera vez que ingresás.
- 🎨 Diseño adaptado a celulares y tablets.

## Deploy en Render.com

Para publicar gratis:

1. Crear cuenta en [Render.com](https://render.com/)
2. Crear nuevo Web Service ➔ Node.js
3. Conectar tu GitHub (o subir manualmente los archivos)
4. Configurar el Start Command:

    ```bash
    node server.js
    ```

5. Variables:
    - PORT: (Render la configura automáticamente)

6. Click en **Deploy** y ¡listo!
